# Morning Star Frontend

React app for video discovery, algorithm tuning, and FTS visualization.

## Setup (Local – Once Git Installed)
1. `npm install`
2. `npm start` → http://localhost:3000 (proxies to core at :8001)

## Features
- Video player with IPFS sourcing
- Real-time tuner (SHAP-integrated)
- FTS chart (polls /trust/score)

Connects to [morning-star-core](https://github.com/YOUR_USERNAME/morning-star-core).
